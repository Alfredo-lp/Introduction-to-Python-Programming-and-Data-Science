Welcome to the Bootcamp

This bootcamp is designed for Northwestern students who have little to none previous experience with programming. Through interactive instruction and project-based work we plan to teach you the basics of programming python and data analysis. To register for the bootcamp use the EventBrite page.

Requirements

You must have a current laptop with Windows or OS X. If it is a Windows laptop then it should have Windows 7, 8, or 8.1 If it is a Mac then it should have OS X 10.9 or 10.10
