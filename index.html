Expectations

We expect that students will attend all days of the bootcamp and complete the assigned homeworks each day. If you must miss a day of the bootcamp, we expect that you will tell us in advance. Also, you must achieve a high-passing score on that day's homework to remain in the bootcamp.

At the end of this bootcamp you should be able/learned to:

  • Write basic Python programs
  • Perform data analysis in Python (plotting, averages, etc.)
  • Basic image analysis
  • Scrape web pages
  • Basic textual analysis
  • Basic structured data analysis

This course is intended as an introduction and to teach you basic programming and data analysis skills. Our goal is for you to acquire a breadth of experience and learn to think programmatically so that you can perform analyses on big data sets on your own/in the future. We are specifically not covering current Big Data technologies.

Syllabus

Day 1. Simple and complex data types, Flow Control

Day 2. Filesystem, File I/O, Functions

Day 3. Imports, Standard Library, Plotting, Review

Day 4. Python Mini-Project, Image Analysis with Numpy

Day 5. Scraping the Web

Day 6. Regular Expressions, Textual Analysis

Day 7. Structured Data Analysis with Pandas
